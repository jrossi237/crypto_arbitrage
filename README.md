# Crypto Arb demo

A Jupytye notebook which identifies a few arb opportunities in the spring of 2018.

## Technologies
This is a juptyer lab notebook, so it requires:
- pandas: Python lib which handles stats and graphs

## Usage

   To run, just type `jupyter labs` in the main directory.
